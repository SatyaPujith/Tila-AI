## ğŸ¯ Key Design Decisions

### 1. **State Management**
- **Local State**: React useState for UI state
- **No Redux**: App complexity doesn't require it
- **Prop Drilling**: Acceptable for this app size
- **Context**: Could be added for global state if needed

### 2. **Auto-Save Strategy**
- **Interval**: 30 seconds (balance between UX and server load)
- **On Message**: Immediate save after each message
- **On Switch**: Save before switching chats
- **Silent**: No notification for auto-save (non-intrusive)

### 3. **Modal System**
- **Centralized**: All modals managed in App.tsx
- **Helper Functions**: Easy-to-use API
- **Type Safety**: Full TypeScript support
- **Reusable**: Same components for all use cases

### 4. **API Design**
- **RESTful**: Standard HTTP methods
- **JWT Auth**: Secure token-based authentication
- **Error Handling**: Graceful fallbacks
- **Validation**: Input validation on both ends
